# Lit
Lit 是一个基于 Web Components 和 Proxy 的 MVVM 框架, 适合公用组件开发<br>
Lit 使用了和 Vue 相似的 API, 大大减少了学习成本

<br>

| :-     | :-      | :-     | :-   | :-                |
| 49+    | 45+     | 10+    | 12+  | No                |
